Hegemony

This README outlines the details of collaborating on this .NET application. This gameis designed to emmulate a game such as democracy!

Prerequisites

You will need the following things properly installed on your computer.

Visual Studio

Installation

git clone https://github.com/Sea413/Hegemony
open with visual studio
Running / Development

.NET server
Visit your app at http://localhost:5000.
Code Generators

Make use of the many generators for code, try ember help generate for more details 

Please download the database SuperScript to run.


To deploy the app you will need to open the project through visual studio. Cloud hosting will come later.
